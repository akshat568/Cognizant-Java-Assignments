# Exercise 15 - ticketraisingapp

## Objective
Create a React App **ticketraisingapp** with `ComplaintRegister` component — a form to raise complaints and get a reference number.

## Component: ComplaintRegister
- Textbox: Employee Name
- Textarea: Complaint description
- `handleSubmit`: generates `REF-XXXXXX` reference number and shows alert

## Run
```bash
cd ticketraisingapp && npm start
```
Open: http://localhost:3000
