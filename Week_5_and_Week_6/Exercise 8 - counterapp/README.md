## Objective
Create a React App **counterapp** with a `CountPeople` component tracking mall entry/exit.

## Component: CountPeople
- **State**: `entrycount`, `exitcount` (stored via constructor)
- **UpdateEntry()**: Increments entrycount when Login button clicked
- **UpdateExit()**: Increments exitcount when Exit button clicked
- Derived: `insideMall = entrycount - exitcount`

## Run
```bash
cd counterapp && npm start
```
Open: http://localhost:3000
