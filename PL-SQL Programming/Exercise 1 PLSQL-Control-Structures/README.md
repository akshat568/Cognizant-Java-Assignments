# Exercise 4: PL/SQL Control Structures

## Objective
Practice PL/SQL control structures using loops, conditions, cursors, and database updates.

## Scenarios Implemented
1. Apply a 1% discount to loan interest rates for customers above 60 years.
2. Promote customers with a balance greater than $10,000 to VIP status.
3. Display reminder messages for loans due within the next 30 days.

## Concepts Used
- DECLARE Block
- CURSOR
- FOR LOOP
- IF Statement
- UPDATE Statement
- COMMIT
- DBMS_OUTPUT.PUT_LINE()
