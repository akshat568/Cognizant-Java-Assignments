# Exercise 17 - fetchuserapp

## Objective
Create a React Application **fetchuserapp** that fetches user data from `https://api.randomuser.me/` and displays title, firstname, and image.

## Component: Getuser
- `componentDidMount()`: async fetch from randomuser API
- Displays: title, firstname, lastname, image, email, phone, location, age, gender
- "Fetch Another User" button re-invokes the API

## API Used
`https://api.randomuser.me/`

## Run
```bash
cd fetchuserapp && npm start
```
Open: http://localhost:3000 (requires internet connection)
