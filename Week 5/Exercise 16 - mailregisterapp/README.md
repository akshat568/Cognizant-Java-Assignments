# Exercise 16 - mailregisterapp

## Objective
Create a React App **mailregisterapp** with a `Register` component validating:
- Name ≥ 5 characters
- Email contains `@` and `.`
- Password ≥ 8 characters

## Validation
- **onChange** (eventhandle): Live validation on each keystroke
- **onSubmit** (eventsubmit): Final validation before form submission

## Run
```bash
cd mailregisterapp && npm start
```
Open: http://localhost:3000
