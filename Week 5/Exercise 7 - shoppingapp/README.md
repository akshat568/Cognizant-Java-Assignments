# Exercise 7 - shoppingapp

## Objective
Create a React Application named **shoppingapp** with class components `OnlineShopping` and `Cart`.

## Classes
- **Cart**: Has `itemname` and `price` properties
- **OnlineShopping**: Creates an array of 5 Cart items, loops through them with `map()` to display a table

## Folder Structure
```
Exercise 7 - shoppingapp/
└── shoppingapp/
    └── src/App.js   ← OnlineShopping + Cart classes
```

## Key Concepts
- Class component with constructor
- Props passing via class properties
- Array `map()` for rendering rows
- `reduce()` for total calculation

## Run
```bash
cd shoppingapp && npm start
```
Open: http://localhost:3000
